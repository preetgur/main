Quickest Way to render template without using the views.py

Inside views.py

First import TemplateView

from django.views.generic import TemplateView

Then set the path for template

urlpatterns = [path("",TemplateView.as_view(template_name = "d_forms/index.html"))]

Assuming we have the App as "d_forms" and then we create a template file in templates folder as "index.html"

That would render our template without views.py

How to send data to TemplateView

First we import some data from database

Inside views.py from blog.models import Blog "Blog" is the name of model all_blogs = Blog.objects.all()

Then send to the template as :

One Way : using "extra_context"
urlpatterns = [path("",TemplateView.as_view(template_name = "d_forms/index.html",extra_context= {'data':all_blogs}))]
No. of blogs : {{data.count}}

Second Way : using simply way
urlpatterns = [path("",TemplateView.as_view(template_name = "d_forms/index.html"),{'data':all_blogs}]
